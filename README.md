# 🤖 ADA — Orientador Profesional Tech (Beta Cerrada)

> **Versión:** 4.1-BETA  
> **Estado:** Experimental (10 usuarios seleccionados)  
> **Motor Lógico:** `ada_questions_beta.json`

Ada es un sistema de orientación profesional basado en IA con una arquitectura de **"Conversación con Propósito Oculto"**. A diferencia de los tests tradicionales de opción múltiple, Ada utiliza técnicas de extracción silenciosa de patrones para definir perfiles técnicos en el sector IT.

## 🎯 Objetivo de la Beta
Validar la fluidez del arco conversacional y la precisión del modelo de recomendación utilizando un set reducido de 7 preguntas de alto impacto, optimizando el tiempo de interacción a menos de 10 minutos.

## 🛠 Arquitectura del Proyecto

El proyecto se divide en tres capas fundamentales:

1.  **Capa de Identidad (`instructions_beta.md`):** Define el sistema de prompts, la personalidad de Ada (mentor senior/amigo dev) y las capas de reacción emocional.
2.  **Capa Lógica (`ada_questions_beta.json`):** Contiene el grafo de conversación, los *signals* de extracción y los *anchors* de intriga.
3.  **Capa de Salida (Resultado Final):** Un motor de resolución que unifica las brechas técnicas con planes de acción a corto, mediano y largo plazo.

## 🚀 Cómo empezar el Test

Para esta beta, se han habilitado 4 disparadores (Conversation Starters) específicos:

* **Bloqueo de Carrera:** *"Siento que sé un poco de todo, pero no me siento bueno en nada. ¿Por dónde empiezo?"*
* **Validación de Stack:** *"Tengo un stack técnico, pero no sé si es el que el mercado está pidiendo hoy."*
* **Escape del "Tutorial Hell":** *"¿Me ayudas a trazar una ruta clara para dejar de saltar de tutorial en tutorial?"*
* **Acceso Directo:** *"Hola, Ada. Tengo 15 minutos de honestidad brutal para mi carrera. ¿Empezamos?"*

## ⚖️ Reglas de Oro de la Interacción
* **Una pregunta a la vez:** Para no saturar el contexto.
* **Honestidad Brutal:** El sistema está diseñado para detectar falta de consistencia.
* **Memoria Visible:** Ada recordará tus respuestas de fases anteriores para conectar puntos.
